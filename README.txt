APA INI?
--------
Bot ini membaca mesej Telegram Web secara automatik melalui Brave browser dan
membalas mesej pelanggan menggunakan AI (Claude). Bot ini berjalan sebagai
akaun anda sendiri — bukan sebagai akaun bot.

KEPERLUAN
---------
1. Windows 10/11
2. Python 3.10 atau lebih tinggi (https://python.org)
3. Brave Browser (https://brave.com)
4. Anthropic API Key

CARA DAPATKAN API KEY ANTHROPIC
--------------------------------
1. Pergi ke https://console.anthropic.com/
2. Daftar atau log masuk
3. Klik "API Keys" di sidebar
4. Klik "Create Key"
5. Salin key tersebut

LANGKAH SETUP (4 LANGKAH SAHAJA)
----------------------------------
1. Buka file .env dan tukar "your_api_key_here" dengan API key anda
2. Klik dua kali START_BRAVE.bat — buka web.telegram.org dan log masuk
3. Klik dua kali START_BOT.bat — bot akan mula berjalan
4. Siap! Bot akan auto-reply mesej yang belum dibaca

APA YANG BOT BUAT
-------------------
✅ Baca mesej masuk yang belum dibaca
✅ Jana reply menggunakan AI berdasarkan produk EA Budak Ubat
✅ Taip dan hantar mesej seperti manusia (bukan copy-paste serentak)
✅ Balas dalam Bahasa Malaysia atau English ikut bahasa pelanggan
✅ Handle group chat (hanya reply jika mesej berkaitan EA/trading)

APA YANG BOT TIDAK BUAT
-------------------------
❌ Tidak buat akaun bot baru
❌ Tidak hantar spam
❌ Tidak balas mesej yang dah dibalas
❌ Tidak crash jika internet terputus

CARA HENTIKAN BOT
-------------------
Tekan Ctrl+C dalam terminal, atau tutup tetingkap terminal.

PENYELESAIAN MASALAH
---------------------
1. "Cannot connect to Brave on port 9222"
   → Pastikan anda telah run START_BRAVE.bat terlebih dahulu

2. "Telegram Web tab not found"
   → Buka https://web.telegram.org dalam Brave dan log masuk

3. "ANTHROPIC_API_KEY not set"
   → Buka .env dan masukkan API key anda

4. "Rate limited by Anthropic"
   → Bot akan tunggu 30 saat dan cuba semula secara automatik

5. "No readable incoming message found"
   → Normal — mesej mungkin gambar/sticker tanpa teks

WHAT IS THIS?
--------------
This bot monitors Telegram Web through your Brave browser and automatically
replies to customer messages using AI (Claude). It runs as YOUR account — not
as a separate bot account.

PREREQUISITES
--------------
1. Windows 10/11
2. Python 3.10+ (https://python.org)
3. Brave Browser (https://brave.com)
4. Anthropic API Key

HOW TO GET ANTHROPIC API KEY
-----------------------------
1. Go to https://console.anthropic.com/
2. Sign up or log in
3. Click "API Keys" in the sidebar
4. Click "Create Key"
5. Copy the key

SETUP (4 STEPS ONLY)
----------------------
1. Open the .env file and replace "your_api_key_here" with your API key
2. Double-click START_BRAVE.bat — open web.telegram.org and log in
3. Double-click START_BOT.bat — the bot will start running
4. Done! The bot will auto-reply to unread messages

WHAT THE BOT DOES
------------------
✅ Reads unread incoming messages
✅ Generates AI replies based on EA Budak Ubat product knowledge
✅ Types and sends messages like a human (character by character)
✅ Replies in BM or English matching the customer's language
✅ Handles group chats (only replies to EA/trading-related messages)

WHAT THE BOT DOES NOT DO
--------------------------
❌ Does not create a new bot account
❌ Does not send spam
❌ Does not re-reply to already answered messages
❌ Does not crash on network failures

HOW TO STOP
------------
Press Ctrl+C in the terminal, or close the terminal window.

TROUBLESHOOTING
----------------
1. "Cannot connect to Brave on port 9222"
   → Make sure you ran START_BRAVE.bat first

2. "Telegram Web tab not found"
   → Open https://web.telegram.org in Brave and log in

3. "ANTHROPIC_API_KEY not set"
   → Open .env and enter your API key

4. "Rate limited by Anthropic"
   → Bot will wait 30 seconds and retry automatically

5. "No readable incoming message found"
   → Normal — the message may be an image/sticker with no text
